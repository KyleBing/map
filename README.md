# 骑行路线规划应用

一个基于 高德地图 API 的 web 应用，可以查看骑行的路线规划。

- vue2
- vue router
- vuex
- 高德 API 2.0

## 一、使用
修改 `/src/mapConfig.js` 中的高德地图开发 key，获取地址： [https://console.amap.com/dev/key/app](https://console.amap.com/dev/key/app)
